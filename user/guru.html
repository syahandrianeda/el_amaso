Dasar Penentuan Nilai Raport di Elamaso

Di elamaso, kami menerapkan sistem pengolahan nilai yang merujuk dokumen Petunjuk Penilaian untuk Sekolah Dasar yang diterbitkan oleh Kemendikbud. Dalam dokumen tersebut kami terjemahkan kedalam algoritma kurikulum 2013.

Adapun dokumen tersebut Anda dapat mempelajarinya berikut ini:
Oleh karena itu, kami melacak data KKM Satuan Pendidikan di sekolah Anda:
KKM Satuan Pendidikan di atas dilacak berdasarkan KKM terendah, jika bukan angka di atas, silakan ubah dan simpan untuk dijadikan Kriteria Predikat di Buku Raport Anda.

Pengolahan Nilai US (Tertulis dan Praktek)

Ketika Anda melihat ini, ada beberapa hal yang harus dipahami. Di antaranya:
  • Saat Anda membuka laman ini, maka Lamaso sedang mengambil data Pengolahan Nilai Server
  • Pengolahan Nilai Server adalah data yang pernah Anda simpan sebelumnya (pernah mengeklik tombol Simpan di Server);
  • Data Pengolahan Nilai Server adalah data yang bisa Anda update langsung (mengubah nilainya) kemudian Anda simpan di server.
  • Ketika Anda belum pernah menyimpan data di server (belum pernah mengeklik tombol Simpan di Server), data yang ditampilkan adalah data Nilai US (Tertulis/Praktek) yang pernah Anda laksanakan di Lamaso. Hal ini sama dengan Anda mengeklik tombol Nilai US Asli
  • Tombol Nilai US Asli menampilkan data Nilai US yang telah diterima Lamaso. Nilai ini adalah nilai Asli siswa Anda sebelum Anda olah/edit. Untuk lebih jelasnya, lihat Status Data pada tabel.
  • Tabel ini tetap menampilkan data US Praktek untuk semua mata pelajaran. Apabila ada mata pelajaran yang tidak diikutsertakan dalam Ujian praktek, maka nilainya akan kosong (tidak ada nilai). Nilai rekap US-nya akan diambil dari Nilai US Tertulis saja.
  • Ada baiknya setelah Anda mengolah data nilai pada tabel ini, Anda wajib menyimpannya di server (dengan mengeklik tombol Simpan di Server). Sebab, data nilai yang disimpan di server (status data: Nilai Pengolahan US) adalah data yang siap untuk dijadikan nilai ijazah di sekolah Anda.
